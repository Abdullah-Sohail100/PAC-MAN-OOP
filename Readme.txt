This game of pac man is built on a 2D integer array.

In int**array following numbers are used to represent following things.

1) 0 is for ball
2) 1 is for wall
3) 9 is for home
4) 8 is for random ghost1
5) 5 is for super cherry
6) 2 is for pacman
7) 3 for empty path
8) 7 is for chasing ghost2
